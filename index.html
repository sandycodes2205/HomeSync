How It Works

From NodeMCU to
your fingertips

HomeSync bridges physical IoT hardware to a beautiful web dashboard via Firebase Realtime Database — no authentication, no friction, just control.

1

NodeMCU sends state

The microcontroller reports device state (ON/OFF) to Firebase over Wi-Fi in real time.

2

Firebase broadcasts instantly

Realtime Database listeners push changes to all open dashboards simultaneously.

3

Dashboard updates live

UI reflects device status, logs the event, and updates timestamps — without any page refresh.
